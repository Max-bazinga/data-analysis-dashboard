# 专业数据分析仪表板

## 🎯 项目概述

一个功能完善、视觉效果专业的数据分析仪表板，支持多种数据可视化、统计分析和导出功能。

**适用场景**: Toptal 作品集 / 数据分析师岗位申请

![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)
![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red.svg)
![Plotly](https://img.shields.io/badge/Plotly-5.18+-green.svg)

---

## ✨ 核心功能

### 📊 数据可视化（6种图表类型）
- **直方图** - 数据分布分析，支持箱线图边缘
- **散点图** - 相关性分析，支持颜色分组
- **箱线图** - 异常值检测和分组比较
- **折线图** - 时间序列趋势分析
- **柱状图** - 分类数据统计
- **饼图** - 占比分析（自动显示Top 10）

### 📈 统计分析
- 描述性统计（均值、标准差、分位数等）
- 相关性矩阵热力图
- 强相关性自动检测（|r| > 0.7）
- 数据质量报告（缺失值、重复行）

### 💾 数据导出
- CSV 格式导出
- Excel 格式导出（多工作表支持）

### 🎨 用户体验
- 响应式侧边栏导航
- 示例数据集（无需上传即可体验）
- 专业配色方案
- 完整的错误处理和用户提示

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行应用

```bash
# 基础版本
streamlit run main.py

# 增强版本（推荐用于作品集）
streamlit run main_enhanced.py
```

### 访问应用

浏览器自动打开：`http://localhost:8501`
